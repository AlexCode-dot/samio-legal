# samio-legal

Public legal documents for the [Samio](https://github.com/AlexCode-dot/samio) iOS app.

## Live URLs

Served via GitHub Pages at `https://alexcode-dot.github.io/samio-legal/`:

- [Privacy Policy (EN)](https://alexcode-dot.github.io/samio-legal/privacy.html)
- [Integritetspolicy (SV)](https://alexcode-dot.github.io/samio-legal/privacy-sv.html)
- [Terms of Service (EN)](https://alexcode-dot.github.io/samio-legal/terms.html)
- [Användarvillkor (SV)](https://alexcode-dot.github.io/samio-legal/terms-sv.html)

## Updating

Edit the HTML files and update the "Last updated" date at the top of each document. Push to `main` — GitHub Pages rebuilds automatically.

Material changes should be announced to users in-app or via email per the policy itself.
